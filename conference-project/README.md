# Optimizing Journal Publications (CRISP-DM Approach)

Presented a machine learning approach to optimizing journal publication outcomes using the CRISP-DM framework. The project focused on identifying key factors that influence publication impact and leveraging predictive modeling to support decision-making.

## Key Highlights
- Applied the CRISP-DM methodology to structure the end-to-end data science workflow
- Performed data collection, preprocessing, and feature selection to identify variables influencing publication impact
- Built and evaluated multiple machine learning models using RapidMiner to predict high-impact outcomes
- Compared model performance using metrics such as specificity, AUC, and gains
- Presented findings at the 4th Annual Graduate Conference for Research, Scholarship, and Creative Activity hosted by The Graduate School at Georgia State University.

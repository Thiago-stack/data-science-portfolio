# Wage Inequality Analysis (IPUMS ACS)

Analyzed over 1.6 million observations from the IPUMS American Community Survey to study the determinants of wage inequality in the United States. The project focused on identifying how demographic, occupational, and geographic factors influence income using statistical modeling techniques.

## Key Highlights
- Worked with a large scale dataset (1.6M+ rows) using Python (pandas) for data cleaning and preprocessing
- Built log-linear regression models to analyze the relationship between education, demographic, occupation, and wages
- Incorporated nonlinear effects and interaction terms to capture complex relationships in the data
- Applied robust standard errors to address heteroskedasticity and improve model reliability
- Interpreted model outputs to identify key drivers of wage variation across populations
